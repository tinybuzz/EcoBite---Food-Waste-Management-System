EcoBite is a frontend-based web platform designed to reduce food waste by connecting users, donors, and organizations. The platform spreads awareness, tracks food waste, and promotes responsible food usage.

## Features
- Food waste awareness dashboard
- Eco-points reward system
- Donation and delivery pages
- User profile management
- Chatbot support
- Responsive UI design

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Type
Frontend Web Application

## How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Navigate to the `pages` folder
4. Open `dashboard.html` in any web browser
